# Besen & Bestien - MVP Walkthrough

## 1. Features
### Login Screen
- Zeigt alle konfigurierten Benutzer (Mock-Daten).
- Unterscheidung zwischen "Dungeonmaster" (Eltern) und Helden.
- Avatar-Placeholder mit Initialen.

### Dashboard (Home)
- Begrüßung mit Name.
- Stats-Übersicht: XP, Schritte, Münzen.
- Fortschrittsbalken für das Wochenziel.

### Navigation
- Tab-Bar mit Icons für:
  - Zuhause (Dashboard)
  - Aufgaben (Platzhalter)
  - Karte (Platzhalter)
  - Held (Platzhalter)

## 2. Testing
Run the app using:
```bash
npx expo start
```
Select 'w' for Web or scan the QR code with your phone.

### Verification Results
- **Build Status:** Success (Web Bundled).
- **Functionality:** App loads on `http://localhost:8081`.
- **Known Issues:** None blocking MVP.
